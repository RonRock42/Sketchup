Responsive Websites

This theme uses a bunch of design techniques that are surprisingly easy to pull of with Responsive Site Designer. Each section consists of a row with background image and single column. The column is transparant but becomes visible on hover. A mini navigation with thumbnail images allows for a cool scroll the different sections.
